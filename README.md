# MotivFit Spor Salonu Web Projesi
**Hazırlayan:** Rabia Demirbaş - 251109046

Merhaba, bu depoda MotivFit Spor Salonu web projemin tüm kaynak kodları ve veritabanı yapısı yer almaktadır.

## Projede Kullandığım Teknolojiler;
* **Görsel Tasarım (Frontend):** HTML, CSS ve EJS
* **Arka Plan Kodları (Backend):** Node.js, Express.js ve MySQL
* **Veritabanı:** MySQL (XAMPP - phpMyAdmin ortamı)

## Projeyi Kendi Bilgisayarımda Çalıştırdığım Adımlar;
Projeyi yerel bilgisayarınızda test etmek isterseniz, şu adımları sırasıyla uygulayabilirsiniz:

1. **Veritabanı Kurulumu:** XAMPP panelinden Apache ve MySQL servislerini başlatın. `phpMyAdmin` ekranına gidip `node_db` adında bir veritabanı açın ve bu depoda paylaştığım `node_db.sql` dosyasını içeri aktarın.
2. **Modülleri Yükleme:** Proje klasörünü terminalde açarak `npm install` komutunu çalıştırın (Böylece gerekli Express ve MySQL paketleri otomatik olarak yüklenecektir).
3. **Sunucuyu Başlatma:** Terminale `node app.js` yazarak sunucuyu ayağa kaldırın.
4. **Tarayıcıda Açma:** Tarayıcınızdan `http://localhost:3000` adresine giderek projeyi inceleyebilirsiniz.
